# personal-site

一个可直接打开的静态个人网站，用来做实习投递展示页。

## 如何使用

1. 编辑 `script.js` 里的 `profile` 对象，把名字、邮箱、项目、技能改成你自己的。
2. 双击打开 `index.html`，就能本地预览页面。
3. 如果要部署到 GitHub Pages、Vercel 或 Netlify，直接上传整个 `personal-site` 文件夹即可。

## 建议优先修改的字段

- `name`
- `title`
- `email`
- `resumeUrl`
- `githubUrl`
- `projects`
- `experience`
- `skillGroups`

## 结构

- `index.html`: 页面结构
- `styles.css`: 页面样式
- `script.js`: 个人信息与动态渲染逻辑
